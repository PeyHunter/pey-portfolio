Projects

My portfolio shows a range of projects with different focuses. I have recently started working with React and have created a live weather app and a fully functional dictionary. Using vanilla html, css and JavaScript I created a live world clock and live weather app. Finally I have worked with Bootstrap inlcuding it in almost all the projects.




react weather app

This is a live weather app that I developed using react and a API. This app is designed to provide users with up-to-date weather information for any location in the world. It includes several features, such as a searchable function Additionally, the app provides a 5 day forecast. To create this app, I used components, props, useState, useEffect and codeMapping among other things.

live world clock

This live world clock utilizes MomentJS , a JavaScript library for timezones. The interface displays current times in various regions, with a current location button, and has a dropdown function and an imported gif. I used html, css to create the design.

vanilla weather app

This is a live weather app that I developed using a weather API and vanilla JavaScript. This app is designed to provide users with up-to-date weather information for any location in the world. It includes several features, such as a searchable function and geolocation button that allows users to quickly find weather information for their location of interest. Additionally, the app provides a 5 day forecast. To create this app, I used html, css using an API and JavaScript to retrieve and display weather information.
